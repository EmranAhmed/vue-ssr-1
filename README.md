# vue-ssr
An implementation of server-side rendering using Vue js and Node. 

## Getting Started
- Clone this repo 
```
git clone https://github.com/samuelayo/vue-ssr
```
- Change directory into the repo
```
cd vue-ssr
```
 - install required packages
``` 
 npm install
```
- Build client bundle
```
npm run build
```
- Build server bundle
```
npm run build-server
```
- Run the application
```
node entry.js
```
